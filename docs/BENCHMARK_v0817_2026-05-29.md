### 2 — EGFR Exon 20 insertion duplicate-key bug fixed (`oncokb_evidence.py`)

**Bug:** Python `dict` last-write-wins. A stale supplemental entry at line ~1051:
```python
("EGFR", "EXON20INS"): {"mobocertinib": "LEVEL_1", "amivantamab": "LEVEL_1"},
```
was silently **overriding** the correct primary entry at line ~124:
```python
("EGFR", "EXON20INS"): {
    "sunvozertinib": "LEVEL_1",
    "amivantamab": "LEVEL_1",
    "poziotinib": "LEVEL_3A",
    "osimertinib": "LEVEL_R1", ...
},
```
The algorithm was presenting withdrawn mobocertinib as LEVEL_1 instead of
sunvozertinib.

**Fix:** Removed the duplicate stale entry. The primary (correct) entry now
takes effect.

**Note on mobocertinib:** Mobocertinib (Exkivity, Takeda) was voluntarily
withdrawn from the US market in November 2023 after the EXCLAIM-2 trial failed
to confirm clinical benefit in the confirmatory phase 3 setting.
